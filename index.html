Introduction

Flavor Notes of NY is a visualization of NYC's urban harvest and is a project generated from the (2016 Treescount! Data Jam)[http://treescountdatajam.devpost.com/]. Using the (2015 NYC Parks treescount dataset)[https://data.cityofnewyork.us/Environment/2015-Street-Tree-Census-Tree-Data/uvpi-gqnh], we seek to sustain public engagement as it relates to urban exploration and local food awareness.

Architecture

The map viz is built in CartoDB with layers for each fruit type. Frontend dev done w/ vanilla JS, some jquery, and (purecss.io)[purecss.io]
